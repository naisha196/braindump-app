### Brain Dump
A simple app to capture and organize ideas in one place

I wanted to start building more often, but I kept losing track of my app ideas because they weren’t stored or curated in one place.
Brain Dump is built to solve that - a single space to quickly capture thoughts and organize them for easy access later.

## Features
* Notes and Ideas categorization - Capture and separate quick notes from larger ideas
* Photo support - Attach images to your entries
* Audio notes - Record thoughts instantly
* All content stored in a pinterest style moodboard

## Improvements in progress
* Semantic search 
* Improved UI and customization
* User authentication

## Future Prospects
* An AI feature to expands on your ideas - 
Suggesting next steps, connections, and improvements
Linking related thoughts into structured concepts
* Cloud sync across devices

## To install on you laptop
* Clone the repository: *git clone https://github.com/your-username/braindump-app.git*
* Navigate into the project on terminal: *cd braindump-app*
* Install dependencies: *npm install*
* Preview the app: *npm start*
* Build the app: *npm run build*
* After you build you will see a dist folder created inside your project folder and the app will be inside this dist folder.
